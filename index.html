Application 1: PO Data Extractor

📄 Problem Statement

Manual processing of bulk PDF purchase orders was time-consuming and error-prone:

  • Creating folders for each request ID
  • Running Python scripts individually
  • Consolidating data using JSON processing
  • Manual data formatting for clients

Application 2: Promo Price Verifier

🔍 Problem Statement

Analyzing promo detection errors was inefficient:

  • Slow image gallery review process
  • Manual validation of pricing rules
  • Difficulty focusing on error patterns
  • Time-consuming error analysis

📏 Built-in Rules

  • MRP = Offer Price + Discount
  • Discount ≤ Offer Price
  • Offer Price ≤ MRP
React.js CSV Processing Rule Engine

📊 Promo Price Verifier Dashboard

✅ Solution Benefits

  • Instant rule-based validation
  • Update custom validation
  • Filtered error-only view
  • Improved QA efficiency
